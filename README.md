# 🚀 Cubana Edge

A modern and interactive landing page for a fictional trading platform built using **HTML**, **CSS**, and **JavaScript**. The project focuses on delivering a premium user experience through smooth scrolling, immersive animations, canvas effects, and responsive UI design.

---



## ✨ Features

- Animated loading screen
- Smooth scrolling using Locomotive Scroll
- GSAP powered scroll animations
- Canvas-based visual effects
- Modern responsive UI
- Interactive product showcase sliders
- Pricing section
- Customer testimonials
- Clean navigation
- Premium landing page design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP
- ScrollTrigger
- Locomotive Scroll
- Remix Icons

## 📚 Libraries Used

### GSAP
Used for high-performance animations and transitions.

### ScrollTrigger
Triggers animations based on the user's scroll position.

### Locomotive Scroll
Provides smooth scrolling and parallax effects.

### Remix Icons
Used for modern UI icons.
